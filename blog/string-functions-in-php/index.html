String Functions in PHP:

A string is a sequence of characters, like "Hello world!". For example, ‘P’ is a character and PHP is a string. In this article we will discuss about strings in details. Every thing inside quotes, single (‘ ‘) and double (” “) in PHP is treated as a string.

1. strlen() - Return the Length of a String

The PHP strlen() function is used to find the length of a string. This function accepts the string as argument and return the length or number of characters in the string.

<?php

echo strlen("Language PHP"); // outputs 12

?>

2. trim() - Count Words in a String

The PHP trim() function allows us to remove whitespaces or strings from both sides of a string.

<?php

echo trim("HP", "Language PHP"); // outputs Language P

?>

3. strpos() - Search For a Text Within a String

The PHP strpos() function takes two string arguments and if the second string is present in the first one, it will return the starting position of the string otherwise returns FALSE.

<?php

echo strpos("Language PHP", "PHP"); // outputs 9

?>

4. str_replace() - Replace Text Within a String

The PHP str_replace() function takes three strings as arguments. The third argument is the original string and the first argument is replaced by the second one. In other words we can say that it replaces all occurrences of the first argument in the original string by second argument.

<?php

echo str_replace("PHP", "C++", "Language PHP"); // outputs Language C++

?>
